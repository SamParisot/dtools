Usage and Information

Though mostly meant for Windows 2 Go installations, we also support PE installations.

For normal use with Windows 2 Go installations or any other installation, you simply double click the executible and change whatever needs to be changed.

PE Support

To run the automatic resolution changer, run "dtools.exe -a"

This automatically finds the highest possible resolution and changes the resolution to that.

Information on each choice

1.) Find Highest Resolution - automatically find the highest resolution

2.) Change Resolution Manually - manually set the resolution

3.) Change Refresh Rate - Change the refresh rate

4.) Change Color Depth - change the color depth

5.) List supported graphics modes - lists avaliable resolutions. refresh rates, and color depths
